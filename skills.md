## Installation Command

To install each skill individually, you can run the following commands:

```bash
bunx --bun skills add

```

To install all skills at once, run the following command:

```bash
bunx --bun skills add https://github.com/hyf0/vue-skills --skill vue-development-guides create-adaptable-composable vue-debug-guides vue-best-practices && bunx --bun skills add https://github.com/antfu/skills --skill vueuse-functions nuxt && bunx --bun skills add https://github.com/coreyhaines31/marketingskills --skill seo-audit && bunx --bun skills add https://github.com/nextlevelbuilder/ui-ux-pro-max-skill --skill ui-ux-pro-max && bunx --bun skills add https://github.com/vercel-labs/agent-skills --skill web-design-guidelines && bunx --bun skills add https://github.com/anthropics/skills --skill frontend-design
```
