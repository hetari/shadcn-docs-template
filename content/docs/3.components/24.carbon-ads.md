The `CarbonAds` component integrates Carbon Ads into your documentation site.

## Usage

```vue showLineNumbers
<script setup lang="ts">
import { CarbonAds } from '@/components/CarbonAds'
</script>

<template>
  <CarbonAds />
</template>
```

## Configuration

Configure your Carbon Ads in the component:

```typescript
const carbonOptions = {
  code: "YOUR_CARBON_CODE",
  placement: "your-placement-name",
}
```

## Features

- Only loads in production (not in dev mode)
- Refreshes on route changes
- Responsive styling
- Customizable appearance

## API Reference

### CarbonAds

No props. Uses internal configuration for Carbon Ads setup.

## Styling

The component includes custom styles for:

- Responsive width
- Background and border styling
- Image and text formatting
- Powered by link styling

## Development

Ads are disabled in development mode (`import.meta.env.DEV`) to avoid loading external scripts during development.
