The `PageHeaderDescription` component provides styled description text for page headers.

## Usage

```vue showLineNumbers
<script setup lang="ts">
import { PageHeaderDescription } from '@/components/PageHeaderDescription'
</script>

<template>
  <PageHeaderDescription>
    A brief description of what this page covers.
  </PageHeaderDescription>
</template>
```

## API Reference

### PageHeaderDescription

| Prop    | Type                    | Default     | Description                |
| ------- | ----------------------- | ----------- | -------------------------- |
| `class` | `HTMLAttributes["class"]` | `undefined` | Custom CSS classes         |

## Styling

- Responsive text sizes: `text-base` (mobile), `sm:text-lg` (desktop)
- Max width: `max-w-3xl`
- Text balance for better wrapping
- Foreground color
